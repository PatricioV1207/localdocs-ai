# LocalDocs AI Quick Guide

LocalDocs AI turns folders of notes, PDFs, and Markdown files into a private searchable knowledge base.

For v0.3, users can upload supported documents, build a local search index, ask questions, generate basic summaries, create flashcards, generate study questions, and export results to Markdown.

The app does not include accounts, authentication, cloud sync, OCR, or a complex vector database in the MVP.
